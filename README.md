# War Card Game

A classic card game implementation in vanilla JavaScript, HTML, and CSS.

## How to Play

1. Click "Draw Card" to draw a card
2. The player with the higher card wins both cards
3. In case of a tie, a "War" happens: each player puts down 3 face-down cards and 1 face-up card
4. The higher face-up card wins all cards in play
5. The game ends when one player has all 52 cards

## Features

- Single player vs Computer
- Automatic shuffling and dealing
- War resolution logic
- Live card count tracking
- Round counter
- Responsive design

## Files

- `index.html` - Game structure
- `styles.css` - Game styling
- `script.js` - Game logic and interaction

## How to Run

Open `index.html` in a web browser. No dependencies or build tools required!
